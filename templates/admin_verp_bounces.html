ℹ️ How VERP Works:
  1. Each outgoing message uses a unique bounce address: sender+recipient_local=recipient_domain@yourdomain.com
  2. If recipient bounces, bounce arrives at the VERP address
  3. GoMail automatically decodes the recipient from the VERP address
  4. Bounce is recorded with original recipient, bounce type, and error code
  5. Admin can view, filter, and analyze bounces per sender
